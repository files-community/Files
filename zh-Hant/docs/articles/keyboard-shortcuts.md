# 鍵盤快捷鍵

Files 支援使用鍵盤快捷鍵以更快速地使用與管理。

| 快捷鍵 | 動作 |
| :---: | :---: |
| `Ctrl` + `C` | 複製 |
| `Ctrl` + `X` | 剪下 |
| `Ctrl` + `V` | 貼上 |
| `Ctrl` + `T` | 開啟新分頁 |
| `Ctrl` + `W` | 關閉分頁 |
| `Ctrl` + `F4` | 關閉分頁 |
| `Ctrl` + `Shift` + `T` | 復原最近關閉的分頁 |
| `Ctrl` + `Tab` | 切換分頁 |
| `Ctrl` + `N` | 開啟新視窗 |
| `Ctrl` + `Shift` + `N` | 新增檔案 |
