# Decisions

This folder contains Architecture Decision Records (ADRs). Use ADRs for
decisions that future maintainers should not have to rediscover from code or
pull request history.

## ADR format

- Status: Proposed, Accepted, Superseded, or Deprecated
- Context
- Decision
- Consequences
